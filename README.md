# Mi configuración de Helix Editor 
https://helix-editor.com/

Para usarlo hay que:
1. Copiar `config.toml` a `~/.config/helix/`
2. Crear una carpeta `themes` en `~/.config/helix/`
3. Copiar monokai_MS.toml a `~/.config/helix/themes/`

Y debería funcionar sin más

**Nota**: El tema sólo es para que cambien de color los modos (normal, insert, select). Si no es necesario, podemos no copiarlo y usar el tema monokai que viene con el editor.
